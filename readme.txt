== Description ==

Change the intervals of automatic updates.

= Changeable =
* Core and their translations
* Plugins and their translations
* Themes and their translations

== Installation ==

1. Upload `update-intervals` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
